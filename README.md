# VentasApi2026

A scalable and modular Sales REST API built with ASP.NET Core 8, designed to manage 
products, inventory, and orders for any type of commerce.

## Tech Stack
- ASP.NET Core 10
- Entity Framework Core + SQL Server
- AutoMapper
- Clean Architecture (Services + Repository Pattern)
- Custom Exception Handling

## Features
- 📦 Product catalog with stock management
- 🛒 Order creation with automatic inventory update
- ✅ Order lifecycle: Pending → Completed / Cancelled
- 📄 Paginated product listing
